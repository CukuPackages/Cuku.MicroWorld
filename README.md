## Installation
1. Open Git Bash / Terminal at the project root
2. Add submodule
   ```
   git submodule add https://github.com/CukuPackages/com.cuku.microworld.git Packages/com.cuku.microworld
   ```

## Dependencies
MicroVerse

MicroSplat

Foliage Renderer

[OsmSharp](https://www.nuget.org/packages/OsmSharp/)
